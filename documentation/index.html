Twitter Account Setup

Goto js folder then open scripts.js file then find Tweetscroll comment then you will find the code like this below .Just change the username envatowebdesign add your own username .

/*----------------------------- Tweetscroll--------------------------*/
  $('.tweets-list-container').tweetscroll({
    username: 'envatowebdesign',
    speed : 100,
    delay: 5000,    
    time: false,
    replies: false,
    limit: 5,
    position: 'append',
    date_format: 'style2',
    animation: 'slide_up',
    request_url: 'js/tweetscroll/twitter/tweets.php',
    visible_tweets: 1
  });
